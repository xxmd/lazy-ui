利用 LazyUI 构建快速、响应式的网站

LazyUI 支持 Sass 变量和 mixin、响应式栅格系统、自带大量组件和众多强大的 JavaScript 插件。基于 LazyUI 提供的功能，能够让你快速设计并定制你的网站。

快速入门 下载 LazyUI

npm安装

使用npm快速安装LazyUI

npm install lazy-ui-webpack

CDN

引入CSS和JS

<!-- 引入CSS -->
<link rel="stylesheet" href="https://unpkg.com/lazy-ui-webpack/dist/css/lazy-ui.min.css"></link>
<!-- 引入JS -->
<script src="https://unpkg.com/lazy-ui-webpack/dist/js/lazy-ui.min.js"></script>
